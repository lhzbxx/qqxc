#接口文档

@model: `User`

----------

###API名称

API名称：`用户修改昵称`

场景说明：`在用户注册初始，用户昵称是系统生成或获取微信授权的微信昵称，用户需要修改现有的昵称。`

请求地址：`/user/modify_nickname`

请求方式：`POST`

----------

###参数说明

__示例__

    {
        'open_id': '7c63edc0-9db1-4894-93c9-23f638ca3b82',
        'nickname': 'qile123456',
        'create_time': '1450261133',
        'verifi': '31C6707C6AA90F5A40DD74C89EBD2EF9'
    }

| 参数 | 说明 |
| ------------- | ------------- |
|open_id|用户OpenID|
|nickname|用户的新昵称|
|create_time|请求创建时间|
|verifi|字典排序后的参数列表的md5加密结果|

----------

###返回值：

__示例__

    {
        'errCode': '100',
        'errMsg': '修改成功'
        'data': ''
    }

| 参数 | 说明 |
| ------------- | ------------- |
|100|修改成功|
|201|该手机号未注册|
|202|昵称已被占用|
|300|参数验证错误|
|301|请求已过期|
|400|OpenID无效|
|500|缺少参数|
